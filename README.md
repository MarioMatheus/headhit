## Requisitos

Uso o [sameboy](https://sameboy.github.io/) para emular o jogo em desenvolvimento.

Após compilar corretamente o jogo, é possível rodar: 

```bash
$ make rodar_jogo
```

## Setup

Baixar o gbdk:

```bash
$ make baixar_gbdk
```
